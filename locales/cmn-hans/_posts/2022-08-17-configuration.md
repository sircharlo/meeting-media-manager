设置界面分为4个部分。 大多数选择都是不言自明的，但这里还有几个补充细节。

### 应用设置

| 设置                     | 说明                                                                                                                                                                                                                                                                                                                         |
| ---------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `离线模式`                 | 如果启用，M³将不会尝试连接到 JW.org 或您的聚会服务器。 当您的网络连接差并想要保存带宽时，这将是有用的。                                                                                                                                                                                                                                                                  |
| `主题设置`                 | 选择您喜欢的主题。 如果您选择`系统`，M³将使用系统主题。                                                                                                                                                                                                                                                                                             |
| `会众名称`                 | 会众名称 这用于支持共享同一计算机的多个会众。                                                                                                                                                                                                                                                                                                    |
| `显示语言`                 | 设置显示M³的语言。 <br><br> 感谢众多贡献者将该应用程序翻译成多种语言！ 如果您想要帮助改进现有的翻译或添加一个新的翻译，请打开一个新的[discussion]({{site.github}}/discussions/new?category=translations)                                                                                                                                                                   |
| `保存媒体的文件夹`             | 聚会媒体将被保存到此文件夹以便以后分享和使用。                                                                                                                                                                                                                                                                                                    |
| `自定义缓存路径`              | 默认情况下，发布和其他数据将保存到每个用户的不同目录。 如果您想要在同一计算机上的多个用户之间共享缓存数据，您可以更改此目录。                                                                                                                                                                                                                                                            |
| `聚会文件夹的日期格式`           | 聚会文件夹使用的日期格式。 <br><br> ***注意：**使用会众服务器时，请确保所有人通过`会众范围设置`强制配置相同的日期格式。*                                                                                                                                                                                                                                          |
| `在系统启动时运行应用`           | 如果启用，M³将在当前用户登录到计算机时启动。 <br><br> ***注意：** Linux 上不可用。*                                                                                                                                                                                                                                                         |
| `自动启动媒体同步`             | 如果启用，此选项将在M³启动5秒后自动启动媒体同步。 <br><br> *为了防止启用此设置时自动同步，请在5秒过去之前按下 ⏸（暂停）键。*                                                                                                                                                                                                                                        |
| `媒体同步后打开文件夹`           | 如果启用，包含所选周下载媒体的文件夹将在媒体同步完成后，在计算机文件管理器中打开.                                                                                                                                                                                                                                                                                  |
| `媒体同步后退出应用`            | 如果启用，该选项将在媒体同步完成5秒后自动退出M³ 。 <br><br> *要防止在启用此设置时 M³ 自动退出，请在5秒过去之前按下 🏃（人离开/奔跑）按钮。*                                                                                                                                                                                                                              |
| `启用OBS Studio兼容模式`     | 如果启用，此选项将与OBS Studio连接，在共享媒体之前和之后按照需要自动改变场景。 <br><br> *如果启用此设置，请确保 OBS Studio 配置为使用`obs-websocket`插件，这将使 M³ 能够与 OBS Studio 进行通信。 <br><br> 另外，配置所有需要的场景用于媒体共享和舞台显示在 OBS 中。 至少，您需要一个场景，其中包含一个配置为捕获 M³ 媒体演示窗口或将呈现媒体的屏幕的`窗口捕获`（推荐）或`显示器捕获`。 <br><br> 您还需要配置所有所需的舞台视图场景，例如：讲台的镜头，舞台的全景镜头等。* |
| `端口`                   | `obs-websocket` 插件被配置为监听的端口。                                                                                                                                                                                                                                                                                               |
| `密码`                   | 在 `obs-websocket` 插件设置中配置的密码。                                                                                                                                                                                                                                                                                              |
| `OBS Studio中的默认舞台视图场景` | 选择启动媒体播放模式时默认选择哪个场景。 通常是舞台宽视图或讲台的镜头。                                                                                                                                                                                                                                                                                       |
| `OBS Studio中的媒体窗口场景`   | 选择在 OBS Studio 中配置的场景来抓取M³媒体窗口。                                                                                                                                                                                                                                                                                            |
| `可选的单独媒体场景图像`          | 一个可选择的场景来选择一个替代场景来显示图像。 如果您想以与视频不同的方式显示图像（例如，使用画中画风格叠加舞台），这将非常有用。                                                                                                                                                                                                                                                          |
| `OBS 工作室场景以显示Zoom参与者`  | 一个可选的场景，用于在混合会议期间快速有效地管理 Zoom 参与者的显示。 <br><br> 当配置此场景时，[媒体播放模式]({{page.lang}}/#present-media)会有一些改变。 在这种模式下，开关按钮会出现，启用时会显示， 将导致媒体窗口被隐藏，并显示缩放场景。 OBS场景选择器也将被隐藏。 共享媒体将会像往常一样自动显示媒体场景，在共享媒体后，媒体窗口将立即消失。 <br><br> 当切换被禁用时，媒体窗口和场景选择器将再次显示。                                                             |
| `启用Zoom内置功能`           | 如果启用，它将尝试实现将各种聚会前的（全部静音、启用王国聚会所的视频/音频、禁用参与者取消静音的选项）、聚会中（选择参与者聚焦并解除静音或设置静音，或放下手并移除聚焦）和聚会后（禁用王国聚会所的视频源，允许参与者解除静音）的操作自动化。 <br><br> ***注意：** 这仍然是一个测试版功能。 请先对它进行彻底测试，总是确认自动化是否正确执行。*                                                                                                                               |
| `Zoom参与者名称`            | 这是M³将用来加入Zoom会议的名称。                                                                                                                                                                                                                                                                                                        |
| `Zoom会议ID`             | M³应加入的Zoom会议的会议ID。                                                                                                                                                                                                                                                                                                         |
| `密码`                   | M³应加入的Zoom会议的会议密码。                                                                                                                                                                                                                                                                                                         |
| `开始会议时聚焦视频`            | 启用时，当执行"开始聚会"自动化时，M³将自动点亮王国聚会所的视频源。                                                                                                                                                                                                                                                                                        |
| `默认隐藏Zoom组件`           | 如果启用，Zoom组件将默认隐藏。 Zoom组件是一个必须的浮动组件，代表Zoom会议的M³参与者。 您可以控制M³参与者。                                                                                                                                                                                                                                                             |
| `自动执行开始聚会操作`           | 如果启用，M³将自动执行"开始聚会"动作，预定会议开始前的已配置的分钟数。 这些行动是：不允许参与者自己静音，禁言所有人，请主机启用视频和麦克风，并可选择地聚焦王国聚会所的视频源。                                                                                                                                                                                                                                 |
| `自动重命名Zoom参与者`         | 通过此设置，您可以添加重命名规则，自动重命名某些Zoom参与者。 如果您想要重命名始终使用错误或不完整名称的参与者，这将是有用的。 规则必须以以下格式输入： `旧名称=新名称`                                                                                                                                                                                                                                   |
| `为测试目的启用测试更新`          | 如果启用，M³将自动收到测试版更新，这些更新每天发布，并包含最新的功能和修复。 <br><br> ***注意：**请勿在用于在王国聚会所播放媒体的计算机上启用此设置。*                                                                                                                                                                                                                           |
| `禁用自动应用更新`             | 启用此选项时，M³关闭后不会自动自动更新。                                                                                                                                                                                                                                                                                                      |
| `禁用硬件加速`               | 只有当您遇到媒体播放模式问题时才启用此设置。 更改此设置将导致M³重启。                                                                                                                                                                                                                                                                                       |

### 会众同步设置

请参阅[会众同步]({{page.lang}}/#congregation-sync)部分，以了解该部分做了什么以及如何配置该部分。

### 媒体设置

| 设置                       | 说明                                                                                                                                                                                                                                                                                                   |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `媒体语言`                   | 选择您的会众或小组的语言。 所有媒体都将用这种语言从 JW.org 下载。                                                                                                                                                                                                                                                                |
| `备份媒体语言`                 | 当主媒体语言不可用时使用此语言。 <br><br> 例如，如果您选择爱尔兰语作为您的媒体语言，英语作为您的备份功能每当没有用爱尔兰语发布出版物或视频时，它将以英语获取。                                                                                                                                                                                                     |
| `视频最大分辨率`                | 从JW.org下载的视频将在本决议中下载，或下一个可用的较低分辨率。 适合有限或低带宽的情况。                                                                                                                                                                                                                                                      |
| `启用视频字幕`                 | 如果您想要获取视频字幕，请启用此选项。 字幕将默认显示，但在显示时可以切换开关。 字幕的垂直位置可以通过单击视频预览中的 `字幕` 图标来更改。                                                                                                                                                                                                                             |
| `将媒体转换为MP4格式`            | 这将自动将所有图片和音频文件转换为 MP4 格式。 用于在完全远程聚合缩放会议期间使用 Zoom的 [原生MP4 共享功能](assets/img/other/zoom-mp4-share.png)。 这包括从JW.org下载的所有图片和媒体文件，以及用户或VO添加的其他媒体文件。 <br><br> ***注意：** 此选项最适合 **远程** 会众Zoom会议。 如果进行**混合**或**常规**会众聚会，查看使用[媒体播放模式]({{page.lang}}/#present-media)激活`显示媒体在外部显示器上或单独窗口`选项，并禁用此选项。*  |
| `转换后保留原始媒体文件`            | 如果启用此设置，图片和音频文件将在转换为 MP4 格式后保留在媒体文件夹中，而不是删除。 这将导致一个略微更细致的媒体文件夹，如果通过Zoom内置MP4共享功能共享媒体，一般不需要启用该功能。 (见 `将媒体转换为 MP4 格式`。) <br><br> ***注意：**仅在`将媒体转换为MP4格式`也启用时才可见。*                                                                                                                          |
| `在外部显示器或单独窗口中播放媒体`       | 此设置将允许您在**混合**或**亲自参与**的会众聚会期间使用M³播放图片、视频和音频文件。 可以通过在M³的主屏幕上点击▶️（播放）按钮来访问媒体播放管理屏幕。 <br><br> 媒体演示屏幕将自动使用外部显示器； 如果不是，媒体将显示在一个单独的、可调整大小的窗口中。 <br><br> ***注意：**此选项最适合于**混合**或**定期**的会众聚会。 <br><br> 如果进行**仅远程**的会众Zoom会议，请考虑激活`将媒体转换为MP4格式`选项，并使用Zoom的原生MP4共享功能共享媒体。* |
| `媒体播放模式的背景图`             | 默认情况下，M³会尝试以先前选择的语言获取当年的全年经文。当[媒体播放模式]({{page.lang}}/#present-media)和没有其他媒体播放时，为了在黑色背景上显示它。 如果由于某种原因无法自动显示全年经文，或者如果您想要显示不同的背景图像， 您可以使用“浏览”按钮选择自定义图片，也可以使用“刷新”按钮尝试再次自动获取全年经文。 <br><br> ***注意：**如果启用了[会众同步]({{page.lang}}/#congregation-sync)选择自定义背景图像将自动同步所有聚会同步用户。*                      |
| `媒体播放结束后隐藏媒体窗口`          | 如果启用，每个媒体文件完成播放后将立即隐藏媒体窗口。 <br><br> ***注意：**此设置特别适用于手语聚会。*                                                                                                                                                                                                                               |
| `自动播放第一个媒体项目`            | 如果启用，媒体列表中的第一个媒体项目将在聚会预定开始前自动开始播放，可配置播放的分钟数。                                                                                                                                                                                                                                                         |
| `在媒体播放时启用键盘快捷键`          | 此设置允许您设置自定义键组合来播放和停止媒体。 例如，这可以与USB遥控器结合在一起。                                                                                                                                                                                                                                                          |
| `创建播放VLC列表`              | 如果您希望自动生成每次会议的播放列表，并且您使用 VLC 来显示媒体而不是 [媒体播放模式]({{page.lang}}/#present-media)，请启用此选项。                                                                                                                                                                                                                 |
| `排除《教导》册子的所有媒体`          | 如果启用，将防止在每次周中聚会上包含*《教导》*册子的媒体内容。                                                                                                                                                                                                                                                                     |
| `排除《永远享受生命》的图像，除非是会众研经班` | 如果启用，这将防止从*《永远享受生命》*（*lff*）中包含的图像。 例如，在中周聚会期间学生练习。                                                                                                                                                                                                                                                   |
| `在可用时包括打印媒体`             | 如果启用，印刷版出版物的图片将在可用时包括在内。 这可能有助于一些以印刷形式更清楚的表格或图像。                                                                                                                                                                                                                                                     |

### 聚会设置

| 设置            | 说明                                                                                                                                              |
| ------------- | ----------------------------------------------------------------------------------------------------------------------------------------------- |
| `特殊会众`        | 如果启用，将不会从JW.org下载媒体。 只有手动添加的媒体才可用。 例如，这对一些训练班可能有用的。                                                                                             |
| `中周聚会`        | 注明周中聚会的日期和时间；用于文件夹命名和自动背景音乐淡出（见下文）。                                                                                                             |
| `周末聚会`        | 注明周末聚会的日期和时间。                                                                                                                                   |
| `下一次分区监督的探访`  | 当您选择下次分区监督的探访日期，M³将自动将该周的中周聚会日期定在星期二。 跳过《会众研经班》媒体，并在两次聚会上跳过王国歌曲。 <br><br> ***注意：**当使用会众服务器时，请确保强制执行此设置，以便所有连接的弟兄们都配置了相同的分区监督探访的日期。* |
| `启用按钮来播放背景音乐` | 启用主屏幕上的按钮，将随机播放*sjm*系列中的王国歌曲。 例如，这有助于在王国聚会所聚会前后播放作为背景音乐的歌曲。 此选项的右边是一个下载所有可用的王国歌曲的按钮，这对于防止缓冲延迟有用。                                                |
| `背景音乐播放音量`    | 设置背景音乐播放的音量。                                                                                                                                    |
| `聚会前自动播放背景音乐` | 如果启用，当是聚会日并且聚会将在不到一小时内开始时，背景音乐将在应用启动时自动开始播放。                                                                                                    |
| `自动停止播放背景音乐`  | 如果`启用播放背景音乐按钮`处于激活状态，那么此设置将允许您指定在多长时间后背景音乐应自动停止。 这可以是设定的分钟数。 或会前预定的秒数（在会前开始背景音乐的情况下）。                                                           |
